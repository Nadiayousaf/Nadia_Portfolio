
# Nadia Yousaf Portfolio

## Overview
This is a **personal portfolio website** featuring a **serverless Python backend** using the **Google Sheets API** to handle contact form submissions.  
The frontend is interactive and modern, including **theme toggle**, **project modals**, **smooth scroll animations**, and a fully responsive layout.

## Features
- **Contact Form**: Messages stored directly in Google Sheets via serverless backend  
- **Theme Toggle**: Switch between dark and light modes  
- **Project Modals**: Interactive project details with tech stack  
- **Smooth Animations**: Scroll reveal for skills, projects, and sections  
- **Responsive Design**: Works on mobile, tablet, and desktop

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript  
- **Backend**: Python serverless function (`/api/contacts.py`) on Vercel  
- **Database**: Google Sheets  
- **Deployment**: Vercel  


## Folder Structure

```
/api             --> Serverless backend
/index.html      --> Frontend main page
/main.js         --> Frontend JS logic
/css             --> Stylesheets
/images          --> Assets
```
